1)In regular functions we need to write the function keyword with the function name
1)In Arrow function we are required thefunction keyword with it it is identified with an arrow operator.

2)Syntax of Regular Function is 
function name {

}
2) Syntax of arrow function is
name = () => {

}

3) While defining regular function we required to write it in curly braces
3) While defining arrow function we are not required to write curly braces if we want to wite only one line of code

4) Regular function can construct objects 
4) Arrow function can not construct objects
